LSTM(Long short-term memory)

先看细节的结构图

图中中间块有4个黄色的框，从左到右f(忘记门) i(输入门1) a(输入门2) o(输出门)，输入门不太严谨，先这样理解。
每个门对应的公式为

公式

F = sigmoid[Wf*(Xt + ht-1) + bf]
I = sigmoid[Wi*(Xt + ht-1) + bi]
A = tanh[Wa*(Xt + ht-1) + ba]
O = sigmoid[Wo*(Xt + ht-1) + bo]
Ct = f*C(t-1) + I*A
Ht = tanh(Ct) O
注意，这里的+和\都表示对应位置加或者乘，不改变矩阵的形状。
各个门的数据在这里应该都是(0,1)的数，忘记门的f和C(t-1)相乘，会变小，可理解为丢弃了一部分信息。而输入门和C(-)相加，会变大，获得新的信息。输出门，将原来的数据和新数据整合，得到新的隐藏数据h。

输入输出细节

下面这张图是我觉得输入细节表现的最好的 来源

对于一个LSTM结构，可以看做由时间序（time_step）个cell构成(实际上只有一个，在循环的过程中，公用一个cell)。每个cell处理一个单词构成的词向量，输出隐藏数据h，对于输出的h,可以将全部用作输出，也可以只取最后一个节点的数据，要看模型的作用。

图中有4个时间序列，可以处理一句话的4个单词，每个单词向量化后是embedding个数字组成的列向量，embedding不固定。

参数计算

若有shape={13,8} 的输入数据，表示将一个含有13个单词的句子，转化为时间序列为13，词向量为8的数据。(将一个词映射为8个数字表示的向量)。这里embedding=8,time_step=13。对T=1时刻，只输入第一个单词对应的向量X1(1,8),若隐藏层有6个，则
对于相关参数的个数有 Wf = (8+6)*6 Bf = 6 该层共90个参数
其他参数个数同样，总共有90*4 = 360个参数，如果最后还有一个全连接层的话，还得加上相应的个数。
